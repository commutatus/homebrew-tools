# homebrew-tools

Homebrew tap for Commutatus tools.

> ⚠️ **Do not make direct changes to this repository.** Formula files are updated automatically by the [cm-tools](https://github.com/commutatus/cm-tools) release workflow on every release.

## Usage

```bash
brew tap commutatus/tools https://github.com/commutatus/homebrew-tools
brew install cm-tools
```
